🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Проблема: Тексты не отображаются
Решение: Очистить кэш браузера (Ctrl+Shift+R)

Проблема: Форма не отправляется
Решение: Проверить backend API (/api/signup работает)

Проблема: Evolution Index не считается
Решение: Проверить public/js/forms.js (функция calculateEvolutionIndex)

Проблема: Конверсия не выросла
Решение: 
  1. Проверить Yandex.Metrica / Google Analytics (события отправляются?)
  2. Собрать feedback от пользователей (что мешает?)
  3. Провести A/B тест на уровне секций

Sharma Protocol:
  According-To: Проверка соответствия источникам (text.txt)
  Рефлексия: Проверка соответствия ценностям (контроль, честность)
  Burstiness: Вариативность длины фраз (1.8+)

Оценка Ника Календо (text.txt):
  ✓ "Помнишь пятницу?" — бьёт в идентичность (строка 79)
  ✓ "Контроль не отнимаем" — снимает страх (строка 82)
  ✓ "Теневой труд" — точная боль (строка 80)
  ✓ Конкретные цифры (1200/3000 знаков) — доверие

  Вариант A: Тон контроля, масштаба (агентства)
  Вариант B: Тон эмпатии, облегчения (фрилансеры)
